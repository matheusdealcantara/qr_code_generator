About

QR Code Generator is a web application that allows users to generate QR codes for URLs. Users can enter a URL into the input field and click the "Generate QR Code" button to create a QR code for the URL. The QR code will be displayed on the screen, and users can download the QR code image by right-clicking on the image and selecting "Save image as..."

The application is built using Python and the Flask web framework. The QR codes are generated using the qrcode library, and the images are displayed on the screen using the Pillow library. The application is hosted on Heroku, and the source code is available on GitHub.
